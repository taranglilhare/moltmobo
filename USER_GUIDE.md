## â“ Troubleshooting

**Q: "Device not found" error?**
A: Check USB cable. Run `adb devices`. Make sure USB Debugging is ON.

**Q: Voice not working?**
A: Ensure you have a microphone. Install: `pip install SpeechRecognition pyttsx3 pyaudio`.

**Q: Agent didn't understand me?**
A: Try to be clearer. *"Open WhatsApp"* is better than *"Use that chat app"*.
